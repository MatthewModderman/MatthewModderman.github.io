
The purpose of this 3D model is to showcase different concepts in software design. 

The concepts shown in this 3D model include:
	- multiple classes
	- multiple instances of the objects
	- phong lighting model
	- encapsulation
	- vbo, ebo, vao for graphics
	- use of dependencies
	- creation and use of functions
	- modular development
	- defensive programming

Interactive user inputs include:
	- L-Alt + M-Mouse Button -> Pan
	- L-Alt + L-Mouse Button -> Orbit
	- F-Key -> Re-initialize Camera
