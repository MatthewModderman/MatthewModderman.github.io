
The purpose of this application is showcase different concepts in NoSQL databases using CRUD. 
The crudprogram.py is the main program which imports and uses all of the other scripts. 

The following concepts can be seen through the application:
	- Creating modular and reusable code
	- User friendly input model
	- Defensive directed developement
	- Create JSON objects to be stored in NoSQL database
	- Read, update, and delete JSON objects stored in NoSQL database
	- Use functions to build value in a business environment
